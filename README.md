# 這是肥貓為了準備轉職考試自己寫的一些輔助程式，基本命名歸則是這樣：
1. 有test的py檔名大致為考試主程式（匯入題庫、考試、算分）；多選題依照台灣金融研訊院的規則，錯一個選項該題就不計分。
2. filehandle的py用意，是彙整當次或歷屆考試考了哪些題目、錯了哪些題；為了方便交叉找解析，所以會秀該題的來源或年份
3. document_handle的用意，除了把考過題目和正確答案print出來之外，還有把所有考古題題目含解答印出來，之後做成隨身聽。這在零碎時間或當床頭音樂助眠應該頗有幫助。
4. ans_handle主要是，如果你題目和解答（析）是放在不同檔案的時候，可以用pandas裡的join功能，自己做key之後直接匯進來，比人工一個一個打快和準。
5. 至於爬取法條的程式，當然也是拿來做隨身聽用的。但我只提供爬開放資料的爬蟲程式。用的時候別亂ping人家伺服器以免觸法。
